Given a fault hypothesis for a connected system in Factorio, generate a repair plan.
The hypothesis includes the component, the affected resource, a description of the fault, and the fault type.
Based on this, determine the specific repair action needed and provide Python code to execute it.

repair options are: reconnect, refuel, repower, replace, rotate, reposition

Available Python actions for repair:
- move_to(position) - Move the agent to a Position(x, y) before interacting with a component
- get_entity(Prototype.entity_type, Position(x, y)) - Get a reference to an entity at a given position
- insert_item(Prototype.item_type, entity, quantity) - Insert an item such as coal or water into a component
- place_entity(Prototype.entity_type, Position(x, y), Direction.UP) - Place a new entity at a position
- rotate_entity(Prototype.entity_type, Direction.RIGHT) - Rotate an entity to correct its orientation

One way to connect entities is to make sure they have the proper rotation. To connect to something face left you should rotate it right.
Generate a single line for the python code such as rotate_entity(get_entity(Prototype.Boiler, Position(x=-5.0, y=-4.5)))

Output the repair plan as a JSON object with repair-description, component, repair, and python-repair fields.
